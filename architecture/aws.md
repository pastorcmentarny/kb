# AWS 

## Services
* EC2 - Elastic Cloud Compute. As service that provides a server.
* SNS - Simple Notification Service  
* IAM - Identy & Access Management 
* VPC - Virtual Private cloud - used for zoning
* AMI - Amazon Machine Image 
* EBS - Elastic Block storage - is an independent storage volume with EC2 instances
* S3 - Simple Storage Service - used to store files
* Load Balancer - routing app that maintains a consistent DNS entry and balances requests to multiple instances
